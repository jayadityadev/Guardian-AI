# Data Schema

## Grooming Dataset
- **columns**: text, label, date_created
- **description**: Contains conversation text samples labeled as grooming or non-grooming

## URL Dataset
- **columns**: url, label, category
- **description**: Contains URLs classified as malicious, phishing, or safe

## Processed Data
- **train.csv**: Training dataset (70%)
- **val.csv**: Validation dataset (15%)
- **test.csv**: Test dataset (15%)
